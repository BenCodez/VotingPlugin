## Features Overview

VotingPlugin offers comprehensive voting functionality for Minecraft servers:

- **Multi-Platform Support**: Works with Spigot, Paper, Bungee, and Velocity
- **Vote Parties**: Community-based voting events with group rewards  
- **Flexible Rewards**: Commands, items, permissions, economy integration
- **Top Voter System**: Leaderboards and recognition for dedicated voters
- **Extensive Configuration**: 870+ configuration options for complete customization
- **Database Support**: SQLite and MySQL with connection pooling
- **GUI Interfaces**: User-friendly interfaces for players and administrators
- **Developer API**: Rich API for plugin developers

## Quick Start
